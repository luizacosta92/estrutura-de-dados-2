# Estrutura de Dados 2

Projeto desenvolvido como resolução de um exercício prático da disciplina de **Estrutura de Dados** (UniSENAI), com foco na modelagem e manipulação de dados de clientes utilizando linguagem C.

## Sobre o projeto

Este repositório apresenta uma implementação modular para gerenciamento de clientes, organizada em arquivos-fonte e cabeçalhos.

A proposta central é aplicar, na prática, os fundamentos de Estrutura de Dados por meio de operações de inserção, organização e manipulação de registros, simulando cenários comuns de sistemas acadêmicos e corporativos.

## Stack utilizada

- **Linguagem:** C
- **Paradigma predominante:** Programação estruturada
- **Ferramentas e recursos:**
	- Compilador C (`gcc`)
	- Biblioteca padrão da linguagem C (`stdio.h`, `stdlib.h`, etc.)
	- Organização em módulos (`.c` e `.h`) para separação de responsabilidades

## Objetivo 

O objetivo deste projeto é servir como **resolução de exercício de Estrutura de Dados**, consolidando conhecimentos essenciais por meio da implementação prática de estruturas e operações fundamentais.

Mais do que apenas produzir o resultado final, a proposta foi exercitar o raciocínio lógico, a construção de algoritmos e boas práticas de organização de código em C.

## Conceitos e conhecimentos aplicados

Durante o desenvolvimento, foram aplicados conceitos importantes da disciplina e da linguagem, como:

- Definição e uso de **estruturas (`struct`)** para representar entidades (clientes)
- Manipulação de **ponteiros** e referências em memória
- Uso de **alocação dinâmica** de memória
- Implementação e manipulação de **nós (nodos)**
- Operações sobre **listas** e estruturas encadeadas
- Separação entre interface e implementação com arquivos **`.h` e `.c`**
- Organização modular do projeto para facilitar manutenção e entendimento
- Práticas de validação e consistência de dados durante as operações

## Resultado esperado

Como resultado, o projeto demonstra a aplicação prática dos conteúdos de Estrutura de Dados, funcionando como material de estudo, revisão e avaliação do aprendizado na disciplina.
